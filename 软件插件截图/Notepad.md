#Notepad++ Json Viewer插件 
<\br>
安装： <\br>
Notpad++ -> 插件(Plugins) -> Plugin Manager -> Show Plugin Manager -> Avaliable -> 选择Json View -> 安装(install) 
<\br>使用： <\br>
选中Json字符串，<\br><\br><\br> 
插件(Plugins) -> Format Json（快捷键Ctrl+Alt+Shift+M） <\br>
插件(Plugins) -> Show Json Viewer  显示Json视图<\br>
